Quantum Stego is a secure steganography platform that allows users to hide secret messages inside images, secured using simulated Quantum Key Distribution. The encryption ensures that even if the stego image is intercepted, the hidden message remains safe without the correct quantum-generated key.

âš™ï¸ How It Works

  1. User uploads an image and enters a secret message.
  2. A quantum key is generated via simulation.
  3. The message is XOR-encrypted with the quantum key.
  4. The encrypted bits are embedded into the image (LSB).
  5. To extract, the image is uploaded along with the key ID.
  6. The system retrieves the key, decodes bits, and decrypts the message.

ğŸ” What If Someone Gets the Image or the Key?

In our system, messages are hidden inside images using steganography and also encrypted using a quantum-generated key. So, what happens if someone intercepts these?

  • If Eve gets the image: It just looks like a regular image. There's no way to tell that it contains a secret message, and without the key, she can't decrypt anything meaningful.
  • If Eve gets the key: That's more serious. If she also has the image, she could extract and decrypt the message. That's why it's important to share the key securely â€” either through an encrypted channel or offline.

In future versions, we can integrate public-key cryptography or even real QKD (Quantum Key Distribution) for sharing keys more securely. But for now, always treat the quantum key like a real password â€” never share it over insecure channels.
